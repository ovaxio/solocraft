# Framework validation produit — solo founder

Adapté au profil : INTP, Learner, dev solo, aversion prospection,
biais vers les systèmes utiles.

## Filtre rapide saas-builder (5 questions)

1. Qui paie ? Particulier, PME, grande entreprise, profession réglementée ?
   -> Préférer PME ou profession libérale : cycle court, budget identifiable

2. Douleur exacte ? Reformuler en une phrase sans jargon.
   -> Si tu ne peux pas en une phrase, le problème n'est pas assez clair

3. Alternatives payantes ? Les concurrents sont une validation, pas un obstacle.
   -> 0 concurrent = marché inexistant ou problème mal défini

4. Distribution évidente ? Comment atteindre les 10 premiers clients ?
   -> Sans canal évident pour un solo sans équipe sales, c'est un risque critique

5. Buildable solo en 4-6 semaines ? Happy path uniquement, pas de rôles multiples.
   -> Si non, découper jusqu'à ce que oui

## Fourchettes de prix marché FR (2026)

| Profil cible | Prix mensuel |
|---|---|
| Outil solo / indépendant | 9-29 €/mois |
| PME / pro libéral | 29-99 €/mois |
| B2B métier niche (notaires, opticiens...) | 79-299 €/mois |
| Lifetime deal lancement | 3-5x le mensuel |

## Signaux de validation réels (par ordre de force)

1. Quelqu'un accepte de payer maintenant -> GO
2. Quelqu'un signe une lettre d'intention -> GO avec conditions
3. Fort intérêt verbal répété -> à valider rapidement
4. "C'est une bonne idée" -> ne vaut rien

## Anti-patterns profil à signaler

- Construire une feature complexe avant d'avoir 1 client
- Pricing trop bas par impostor syndrome
- Aucun canal de distribution identifié
- Feature qui demande de la prospection froide intensive
